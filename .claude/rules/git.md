# Git Rules

- Conventional Commits: `feat:`, `fix:`, `docs:`, `test:`, `chore:`, `refactor:`
- Commit message: imperative mood, англійська, ≤72 символи у першому рядку
- Тільки feature branches: `feat/...`, `fix/...`, `chore/...`
- Жодних прямих комітів у `main`
- PR обов'язковий перед merge
- Merge тільки після зеленого CI
