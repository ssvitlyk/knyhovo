# packages/shared

Shared типи, domain entities та утиліти. Єдине джерело правди для типів у монорепозиторії.

## Що тут живе

- TypeScript типи та інтерфейси domain: `Book`, `CanonicalBook`, `ProviderListing`, `WishlistItem`, `PriceHistory`
- `ScraperProvider` інтерфейс (реалізується у `packages/scrapers`)
- Спільні утиліти (нормалізація рядків, форматування цін тощо)

## Що тут НЕ живе

- Бізнес-логіка з побічними ефектами (HTTP, БД)
- UI компоненти
- Env-specific конфігурація

## Правило

Якщо тип потрібен у двох або більше пакетах — він живе тут.
Якщо тип специфічний для одного пакету — він живе у цьому пакеті.
