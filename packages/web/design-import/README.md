This folder is for temporary and archived Claude Design exports.

## incoming/

Temporary design packages copied from Claude Design.

These files are not the source of truth for implementation.

## archive/

Historical design packages kept for reference and audits.

## Source of truth

Claude Code should use these as implementation sources instead:

- docs/prd/*

- approved implementation plans

- frozen specs explicitly referenced by the current task

Do not implement directly from stale design-import packages unless the task explicitly says so.
